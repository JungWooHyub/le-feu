# 게시글 작성/수정/삭제 기능

**Task ID:** T-015
**Status:** BACKLOG
**Importance:** MUST
**Complexity:** 7/10
**Urgency:** 9/10
**Dependencies:** T-014, T-011

## Description

### 설명
텍스트 및 이미지 첨부가 가능한 에디터로 게시글 작성·수정·삭제 기능 구현

### 구현 세부 사항
1. 'pages/community/new.tsx'와 'pages/community/[id]/edit.tsx' 컴포넌트 생성
2. React Quill 또는 Editor.js 기반 리치 텍스트 에디터 통합
3. Cloudinary 이미지 업로드 연동 및 에디터 내 이미지 삽입 로직 구현
4. 작성·수정 API 라우트 생성(API Routes)으로 Supabase 'posts' 테이블 CRUD 구현
5. 삭제 버튼 클릭 시 확인 모달 후 API 호출 및 상태 업데이트
6. 권한 체크: 작성자와 로그인 사용자 비교 후 UI 토글

### 테스트 전략
- 에디터 텍스트/이미지 삽입 후 게시글 생성 테스트
- 수정 화면 로드된 데이터 반영 및 업데이트 테스트
- 삭제 확인 모달 및 취소/확인 시나리오 검증
