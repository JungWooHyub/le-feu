## Description

### 설명
게시글 하단 실시간 댓글 및 대댓글 작성·수정·삭제 시스템 구현

### 구현 세부 사항
1. Supabase Realtime 채널 구독 및 'comments' 테이블 변경 이벤트 리스닝 설정
2. 'components/CommentList.tsx'로 댓글/대댓글 트리 구조 렌더링
3. 입력창 컴포넌트에서 댓글 작성/수정/삭제 API 호출 구현
4. 익명/실명 토글(옵션) 및 작성자 역할/닉네임 표시 로직 추가
5. 실시간 이벤트 수신 시 상태 업데이트로 UI 자동 동기화
6. React Query 또는 SWR로 캐싱 및 옵티미스틱 업데이트 처리

### 테스트 전략
- Realtime 채널 테스트용 모킹으로 실시간 동기화 확인
- 작성·수정·삭제 API 응답에 따른 UI 변경 검증
- 익명/실명 토글 기능 시나리오 테스트
