# 커뮤니티 메인/카테고리별 게시판 UI

**Task ID:** T-014
**Status:** BACKLOG
**Importance:** MUST
**Complexity:** 6/10
**Urgency:** 9/10
**Dependencies:** T-013

## Description

### 설명
커뮤니티 메인 및 카테고리별 게시판 리스트 UI와 페이지네이션 구현

### 구현 세부 사항
1. 'pages/community/index.tsx'와 'pages/community/[category].tsx' 생성
2. Supabase에서 질문/후기/자유/구인 등 카테고리별 게시글 목록 쿼리 구현
3. 게시글 카드 컴포넌트에 썸네일, 제목, 요약, 작성자, 날짜 표시
4. Tailwind CSS로 반응형 그리드 레이아웃 구성 및 모바일 우선 디자인 적용
5. 페이지네이션 컴포넌트 구현 및 Prev/Next 버튼 상태 관리
6. 카테고리 탭 클릭 시 URL 쿼리 및 SSR/SSG 방식으로 데이터 패칭

### 테스트 전략
- 각 카테고리별 데이터 API 모킹 및 렌더링 확인
- 페이지네이션 버튼 동작 테스트(엔드 투 엔드)
- 모바일/태블릿/데스크톱 뷰포트에서 레이아웃 검증
