**Status:** BACKLOG

## Description

### 설명
게시글 및 댓글별 좋아요·북마크 기능과 내 모아보기 UI 구현

### 구현 세부 사항
1. Supabase 'likes' 및 'bookmarks' 테이블 설계 및 마이그레이션 적용
2. 게시글·댓글 카드와 상세 화면에 좋아요·북마크 버튼 추가
3. 버튼 클릭 시 API 호출로 상태 토글 및 카운트 업데이트
4. 사용자 개인 페이지 'pages/profile/[userId]/favorites.tsx'에 좋아요·북마크 모아보기 리스트 구현
5. UI 토글 시 로딩 스피너 및 에러 핸들링 추가

### 테스트 전략
- 좋아요·북마크 버튼 클릭 후 DB 상태 변경 모킹 검증
- 개인 즐겨찾기 페이지에서 데이터 렌더링 확인
